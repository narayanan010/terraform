# EKS running config reference for Terraform module 'eks-onboading'

This is an example of how to use 'eks-onboarding' Terraform module that will create new k8s resources within EKS cluster. The EKS settings should be imported from an existing EKS cluster.

Following running-config folders structure are examples for reference purpose only

```
├── sandbox_example      # Sample of running-config for `sandbox` environment
│   ├── cluster
│   └── sample
└── staging_example      # Sample of running-config for `staging` environment
```

This running-config for staging is for reference purpose only

&nbsp;
### Running-config references

- sandbox_example: Review [README.md](./sandbox_example/README.md) from `sandbox_example` for extended details
- staging_example: Review [README.md](https://github.com/capterra/terraform/tree/master/modules/eks-onboarding/README.md) from `eks-onboarding` common running details.
